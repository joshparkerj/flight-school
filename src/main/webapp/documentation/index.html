Introduction

Flight School API has all the information you could want about Flight School™, a totally legitimate institution of higher-learning. This documentation should help you familiarize yourself with the resources available and how to consume them with HTTP requests. If you have problems, all I can say is just keep trying things and never give up!

Getting started

Let's make our first API request to the Flight School API!

Just paste the url for whatever resource you want into your browser address bar! It is also possible to use curl, or a great API tool like Postman. Personally, I recommend using the Google Chrome browser with the JSONView extension installed. It makes browsing our API a breeze!

Try visiting the API endpoint for the Boeing 747:

http://flightschool.joshquizzes.com/flightschool/api/aircraft/14/

Base URL

The Base URL is the root URL for all of the API. It is https://flightschool.joshquizzes.com/flightschool/api/ The documentation below assumes you are prepending the Base URL to the endpoints in order to make requests.

Authentication

Flight School™ provides a completely open API. No authentication is required to query and get data. This also means that you are limited to GET requests. If you find a mistake in the data, feel free to laugh at our mistake and feel very superior. There's no charge for schadenfreude!

JSON Schema

All resources support JSON Schema. Making a request to /api/<resource>/schema will give you the details of that resource. This will allow you to programmatically inspect the attributes of that resource and their types.

Searching

All resources support a search parameter that filters the set of resources returned. This allows you to make queries like: https://flightschool.joshquizzes.com/flightschool/api/pilot/?search=maverick .

All searches will use case-insensitive partial matches on the resources name.

Encodings

JSON

All api responses will be JSON-formatted.

Resources

Root

The Root resource provides information on all available resources within the API.

Pilots

The Pilot resource provides information on all of the pilots who have totally earned their certifications from Flight School™.

Certifications

The Certification resource lists all of the completely legitimate certifications that we award to the dedicated pilots who have graduated from Flight School™.

Aircraft

The Aircraft resource lists all of the aircraft which our completely capable and reliable pilots can be certified to fly.
